# Jad Law Firm Website

موقع ويب مستقل لمكتب جاد للمحاماة والاستشارات القانونية.

## التشغيل المحلي

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

ثم افتح:

```text
http://127.0.0.1:4173
```

## الملفات

- `index.html`: هيكل الموقع.
- `styles.css`: التصميم والاستجابة للجوال.
- `script.js`: المحتوى العربي والإنجليزي، الصفحات الداخلية، وتبديل اللغة.
- `assets/logo.png`: شعار المكتب.
- `assets/founder.png`: صورة قيادة المكتب المتاحة للاستخدام.
- `assets/office/`: صور المكتب المستخدمة في الهيرو والمعرض وأقسام التواصل.
- `assets/fonts/`: خط Thmanyah Serif المستخدم للعناوين.
- `assets/legal-documents/cases/`: ملفات الأحكام والقضايا الأصلية كما تم تزويدها.
- `assets/legal-documents/license/`: رخصة مزاولة مهنة المحاماة ومعاينتها.

## لوحة التحكم (Admin Panel)

يحتوي المشروع الآن على لوحة تحكم محلية لتعديل محتوى الموقع بدون فتح الأكواد. راجع تفاصيل التشغيل والاستخدام الكاملة في ملف `ADMIN_GUIDE.md`.

تشغيل سريع:

```powershell
npm install
npm run admin
```

ثم افتح `http://localhost:3000/admin` من نفس الجهاز فقط.
